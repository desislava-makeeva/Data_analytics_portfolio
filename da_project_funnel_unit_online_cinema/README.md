# Анализ убыточности маркетинга и юнит-экономики онлайн-кинотеатра

**Ссылка на проект:** [https://docs.google.com/spreadsheets/d/1uEHsxrMBgpBxcNpOC9pNrum8RIJJSPS1J437XZrMFnY/edit?gid=302097316#gid=302097316]

**Цель проекта:** 
Выявить причины убыточности рекламных кампаний и предложить сценарии для вывода юнит-экономики в плюс.

**Задачи:**
- Проанализировать динамику основных метрик (выручка, траты, CAC, ROI) в течение года.
- Провести воронку продаж в разрезе рекламных каналов для поиска точек роста.
- Выполнить когортный анализ: оценить удержание (Retention) и LTV.
- Построить финансовую модель для тестирования гипотез по улучшению ключевых показателей.

**Инструменты и ключевые слова:** Google Sheets (сводные таблицы, диаграммы, формулы массива, когортный анализ), unit-экономика, пользовательская воронка, CJM.

# Анализ рынка недвижимости Санкт-Петербурга и Ленинградской области

**Автор:** Десислава Макеева  
**Инструменты:** Yandex DataLens, SQL  
**Дашборд:** [Смотреть онлайн](https://datalens.yandex/4r8bj0bm6024q)

---

## 📌 О проекте

Исследование рынка недвижимости двух регионов на основе данных объявлений о продаже. Цель — выявить факторы, влияющие на скорость продаж, сезонность и региональные особенности для формирования рекомендаций участникам рынка.

---

## 🔍 Ключевые вопросы исследования

- Какие сегменты недвижимости продаются быстрее/медленнее?
- Как характеристики объектов (площадь, цена, инфраструктура) влияют на сроки продажи?
- Чем отличается рынок Санкт-Петербурга от Ленинградской области?
- Какова сезонность на рынке недвижимости?

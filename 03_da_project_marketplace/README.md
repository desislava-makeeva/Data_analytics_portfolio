# Анализ данных маркетплейса: аудит метрик и A/B-тестирование

## 📌 Описание проекта
Комплексный анализ данных маркетплейса, включающий аудит качества данных, исследование ключевых бизнес-метрик, когортный анализ и оценку результатов A/B-теста по увеличению конверсии.

## 🎯 Ключевые задачи
- Сбор, обработка и оценка качества данных
- Анализ ключевых метрик: выручка, затраты, количество заказов, MAU/DAU, ARPU, ARPPU, Revenue rate
- Когортный анализ монетизации и юнит-экономики (ROI, LTV, CAC) по каналам привлечения
- Выявление проблемных этапов воронки продаж
- Оценка эффективности A/B-теста (бонус за первую покупку)

## 🔍 Основные результаты

**Аналитика и метрики:**
- Проведен аудит качества данных, выявлены и обработаны аномалии
- Проанализирована динамика DAU/MAU и распределение пользователей по частоте покупок
- Рассчитаны метрики монетизации и юнит-экономики в разрезе когорт и каналов трафика
- Определен канал с пониженной конверсией на ключевых шагах воронки

** Анализ результатов A/B-тестирования:**
- **Гипотеза:** бонус за первую покупку повысит конверсию из регистрации в покупку среди пользователей, привлеченных из TikTok
- **Результат:** 
  - Конверсия в экспериментальной группе: **7.5%**
  - Конверсия в контрольной группе: **1.1%**
  - Статистическая значимость: p-value < 0.05

## 🛠 Используемые инструменты
- **Python:** pandas, numpy, scipy (статистические тесты), matplotlib/seaborn (визуализация)
- **SQL:** выгрузка и агрегация данных
- **Jupyter Notebook:** документирование и презентация результатов
